# Superstore Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes a Superstore sales dataset to provide insights into business performance, including sales trends, top categories, regional profits, and overall profitability. The dashboard is built in **Power BI**.

## 🛠️ Tools Used
- Power BI Desktop
- DAX Measures

## 📂 Dataset
- Superstore Sales Dataset (Kaggle)
- Columns used: Order Date, Sales, Profit, Category, Region, Order ID

## 📈 Key Metrics / KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin (14%)

## 📊 Insights & Recommendations
- Technology category generates the highest revenue → consider expanding product line
- East region is the most profitable → focus marketing there
- Sales peak during February → increase promotions during peak months
- Overall profit margin is 14% → evaluate cost-saving opportunities
